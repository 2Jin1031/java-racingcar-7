## 자동차 경주

---

### 구현할 기능 목록

입력
- 자동차 이름 입력
  - 쉼표(,)를 기준으로 구분
  - 각 자동차 이름
    - 5자 이하만 가능
    - 공백 불가능
    - 중복 불가능
- 시도할 횟수 입력
  - 1 이상의 정수만 가능


출력
- 횟수번 시도 결과
  - 각 자동차 이름
    - 이름 출력
    - 현재 위치 출력
      - 현재 위치에 따른 '-' 출력
  - 각 시도 결과 출력
    - 랜덤 값에 따른 전진 여부 출력
        - 4 이상일 경우 전진
        - 3 이하일 경우 정지
        - 전진 시 위치 1 증가
- 최종 우승자 출력
  - 가장 멀리간 자동차 이름 출력
  - 우승자 여러명일 경우 쉼표(,)로 구분하여 출력
